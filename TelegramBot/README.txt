Пример для заполнения файла config.ini
[Telegram]
token = ваш_токен

[API]
url = ссылка_на_API

Ниже я приведу пример как это сделать:

[Telegram]
token = 1234567890:abcdefghijklmnopqrstuvwxyz

[API]
url = https://api.exchangerate-api.com/v4/latest/






В данном примере "token" является токеном вашего телеграм-бота  
url - ссылкой на API, которая будет использоваться в вашем скрипте.
